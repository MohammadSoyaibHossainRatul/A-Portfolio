Vanilla Java Script

Vanilla JS is the plain JavaScript language, without any additional frameworks or libraries. It's the core foundation for web development, providing the building blocks for creating interactive and dynamic web applications.

React

React is a JavaScript library for building user interfaces. It uses a component-based approach to create reusable and efficient code, making it ideal for complex web applications.

Node JS

Node.js is a JavaScript runtime environment that allows you to run JavaScript code outside of a web browser. It's popular for building server-side applications, APIs, and command-line tools.

Mongo DB

MongoDB is a popular NoSQL database that stores data in flexible JSON-like documents. It's known for its scalability, performance, and ease of use, making it ideal for modern applications.
